Demystifying ::before

The ::before inserts content (specified in CSS) before the content found inside the element in which you intend to insert said content. These can also be stylized further.

We have seen in the title that the words "HTML & CSS" have been set within CSS to appear before the content in the h1 tag. It is of paramount importance that you add the content property within the [element]::before ruleset otherwise the changes within said ruleset will simply be ignored. Note that you do not have to add text, thus you can leave the quotes empty.

content: "";

Anyways, you can use other styling properties such as padding, background-color (or background...), color, whatever you want! :)

Demystifying ::after

Works akin to ::before, except content is placed after the content within the element you wish to stylize using ::after.
